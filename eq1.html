Q1: Write the sets in roster form

(i) A = {x : -3 ≤ x < 7 and x is an integer}

(ii) B = {x : x is a prime number that divides 60}

Solution:

(i) The set A contains integers between -3 and 7 (excluding 7):

A = {-3, -2, -1, 0, 1, 2, 3, 4, 5, 6}

(ii) The prime factors of 60 are:

60 = 2 × 2 × 3 × 5. Hence, B = {2, 3, 5}

Q3: Find A - B and A' ∪ B'

(i) A - B: Elements in A but not in B.

(ii) A' ∪ B': Elements outside both sets A and B.

Solution:

(i) A - B = {3, 6, 9, 15, 18, 21}

(ii) A' ∪ B': Complement of A ∪ B, i.e., elements outside A and B.

Q4: Find complements of sets

(i) X = even natural numbers

(ii) Y = {x : x + 5 > 8}

Solution:

(i) Complement of X = odd natural numbers

(ii) For Y = {x : x > 3}, the complement is {x : x ≤ 3}

Q5: Find A × A if A = {1, -1}

Solution:

Cartesian product A × A gives all ordered pairs:

{(1, 1), (1, -1), (-1, 1), (-1, -1)}

Q6: Let R be a relation on A = {2, 3, 4, 6}
