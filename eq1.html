Q1: Write the sets in roster form

(i) \( A = \{x : -3 \leq x < 7 \text{ and } x \text{ is an integer} \}

(ii) \( B = \{x : x \text{ is a prime number that divides 60}\} \)

Solution:

(i) \( -3 \leq x < 7 \): Iska matlab hai x ek integer hai jo -3 aur 7 ke beech mein aata hai.

Set \( A \): {-3, -2, -1, 0, 1, 2, 3, 4, 5, 6}

(ii) \( B \): Prime numbers jo 60 ko divide karte hain:

60 = \( 2 \times 2 \times 3 \times 5 \), toh primes hain {2, 3, 5}

Q2: Write the sets in set-builder form

(i) \( A = \{2, 4, 8, 16, 32\} \)

(ii) \( B = \{1, 4, 9, 16, 100\} \)

Solution:

(i) \( A = \{x : x = 2^n, n \in \mathbb{N}, 1 \leq n \leq 5\} \)

(ii) \( B = \{x : x \text{ is a square of a natural number}\} \)

Q3: Find \( A - B \) and \( A' \cup B' \)

(i) \( A - B \): Elements jo \( A \) mein hain but \( B \) mein nahi.

(ii) \( A' \cup B' \): Area outside \( A \cup B \).

Solution:

(i) \( A - B = \{3, 6, 9, 15, 18, 21\} \)

(ii) \( A' \cup B' \): Complement of \( A \cup B \), which is outside both sets.
