Solutions

Q1. Write the sets in roster form

(i) \( A = \{x : x \text{ is an integer and } -3 \leq x \leq 7\} \)

Solution: \( A = \{-3, -2, -1, 0, 1, 2, 3, 4, 5, 6, 7\} \)

(ii) \( B = \{x : x \text{ is a prime number which is a divisor of 60}\} \)

Solution: \( B = \{2, 3, 5\} \)

Q2. Write the sets in set-builder form

(i) \( P = \{2, 4, 8, 16, 32\} \)

Solution: \( P = \{x : x = 2^n, n \in \mathbb{N}, 1 \leq n \leq 5\} \)

(ii) \( Q = \{1, 4, 9, 16, 25, 49, 81, 100\} \)

Solution: \( Q = \{x : x = n^2, n \in \mathbb{N}, 1 \leq n \leq 10\} \)

Q3. Draw the Venn Diagram

(i) \( A - B \) = Elements in \( A \) not in \( B \)

(ii) \( A' \cup B' \): Universal set ka use karte hain.

Solution: \( A - B = \{3, 6, 9, 15, 18, 21\} \)

Q4. Find \( A \times A \) if \( A = \{1, -1\} \)

Solution: \( A \times A = \{(1, 1), (1, -1), (-1, 1), (-1, -1)\} \)

Q5. Complement of sets

(i) \( X = \{x : x \text{ is an even natural number}\} \)

Solution: \( X' = \{1, 3, 5, 7, 9, \ldots\} \) (Odd numbers)

(ii) \( Y = \{x : x + 5 > 8\} \)

Solution: \( Y = \{4, 5, 6, 7, \ldots\} \)

Q6. Let \( A = \{\frac{2}{3}, \frac{5}{3}\} \)

Solution: \( x + y = \frac{4}{3} \), \( x - y = 0 \)

Q7. Relation R on A

Set \( A = \{2, 3, 4, 6\} \)

Solution: \( R = \{(2, 2), (2, 4), (2, 6), (3, 3), (3, 6), (4, 4), (6, 6)\} \)
